# Tip Calculator

A simple tip calculator and bill splitter built with HTML, CSS, and JavaScript.

## How to Run

Just download the file and open it in any browser. No installation needed.

```
open index.html
```

## What it does

- Enter a bill amount in Rs
- Pick a tip percentage (10%, 15%, 20%) or type a custom one
- Enter number of people
- See the tip amount, grand total, and per person share — updates as you type
- Shows error messages if you enter invalid values
- Reset button to clear everything
